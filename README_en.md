# PKtraduction

![PKtraduction](store/icon.png)

[🇬🇧 EN](README_en.md) · [🇫🇷 FR](README.md)

Instantly translate text with popup display or direct text substitution.

## ✅ Features
- Instant translation of selected text.
- Direct text substitution mode.
- Full page batch translation.
- Real-time loading indicator.
- Minimalist interface and automatic positioning.

## 🧠 Usage
- **Popup translation:** Select any text on a webpage, the popup displays automatically below it.
- **Direct substitution:** Enable the option in settings, the selected text will be replaced.
- **Full page translation:** Click the extension icon to translate all content on the active page.

## ⚙️ Settings
- Target language (default: French).
- Auto-translate mode.
- Direct substitution option.

## 🧾 Commands
- Click extension icon: trigger full page translation.
- Text selection: trigger quick translation.

## 📦 Build & Package
- Source files are located in `/src`.
- Assets are located in `/store`.
- Compressed build is available in `/release/extension.zip`.

## 🧪 Install (Antigravity)
1. Download or clone this repository.
2. Open `chrome://extensions/` in Chrome.
3. Enable **Developer Mode**.
4. Click **Load unpacked** and select the `/src` folder.
