Politique de confidentialité

PKtraduction Chrome Extension

Dernière mise à jour : 26 juin 2026
Cette extension traite uniquement des données nécessaires à son fonctionnement local. Elle ne collecte pas de données sensibles vers nos serveurs.

Données traitées

L'extension peut traiter localement le texte sélectionné par l'utilisateur afin d'effectuer la traduction demandée. Les paramètres de l'extension sont stockés localement dans Chrome via `storage`.

Utilisation des données

Les données sont utilisées uniquement pour fournir les fonctions de traduction, d'affichage popup, de substitution directe et de configuration de l'utilisateur.

Stockage et transfert

Les données restent sur l'appareil de l'utilisateur ou dans le stockage géré par Chrome. L'extension n'envoie pas d'informations personnelles vers un serveur propriétaire.

Partage et transfert

Nous ne vendons pas, ne louons pas et ne partageons pas les données utilisateur avec des tiers, sauf si cela est strictement nécessaire pour fournir la traduction demandée via les services de Google utilisés par l'extension.

Cookies et suivi

L'extension n'intègre pas de cookies, SDK publicitaire ou mécanisme de suivi comportemental.

Accès et suppression

L'utilisateur peut effacer les données stockées localement en supprimant les données de l'extension depuis Chrome ou en désinstallant l'extension.

Permissions Chrome

  • activeTab : pour agir sur l'onglet actif quand l'utilisateur déclenche une traduction.
  • storage : pour conserver les préférences locales.
  • scripting : pour injecter le script nécessaire à la traduction et à l'affichage.
  • contextMenus : pour proposer des actions rapides dans le menu contextuel.

Modifications de cette politique

Cette politique peut être mise à jour si l'extension évolue. La version publiée sur ce site est la référence la plus récente.

Contact

Pour toute question, contactez contact@mondary.design.

English Version

Data Processed

The extension may process locally the text selected by the user to perform the requested translation. Extension settings are stored locally in Chrome through `storage`.

Use of Data

Data is used only to provide translation, popup display, direct substitution, and user configuration features.

Storage and Transfer

Data stays on the user's device or in Chrome-managed storage. The extension does not send personal information to a proprietary server.

Sharing and Transfer

We do not sell, rent, or share user data with third parties, except where strictly necessary to provide the requested translation through the Google services used by the extension.

Cookies and Tracking

The extension does not use cookies, ad SDKs, or behavioral tracking mechanisms.

Access and Deletion

Users can erase locally stored data by removing extension data from Chrome or by uninstalling the extension.

Chrome Permissions

  • activeTab: to act on the active tab when the user triggers a translation.
  • storage: to keep local preferences.
  • scripting: to inject the script needed for translation and display.
  • contextMenus: to expose quick actions in the context menu.

Changes to This Policy

This policy may be updated if the extension evolves. The version published on this site is the latest reference.
