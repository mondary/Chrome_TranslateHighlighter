# PK Traduction - Description Chrome Web Store

## 🇬🇧 English

Instant translation with popup display. Made by PK-Labs.

Quick translation tool with popup or direct text substitution. Translate selected text instantly using Google Translate API.

**Features:**
- Translate selected text via right-click
- Popup display for quick preview
- Direct text substitution option
- Supports multiple languages
- Works on any website
- Fast and accurate translation

Perfect for reading foreign content, international research, and anyone who works with multiple languages.
